# 🏥 Hospital Emergency Dashboard using Microsoft Excel

## 📌 Project Overview

The Hospital Emergency Dashboard is an interactive Excel dashboard developed to analyze emergency department performance and patient data. It provides meaningful insights through KPIs, charts, Pivot Tables, and slicers, helping healthcare professionals monitor hospital operations and improve decision-making.

---

## 🎯 Objectives

- Analyze emergency patient records
- Monitor patient admissions
- Track waiting time
- Understand patient demographics
- Analyze department-wise cases
- Improve hospital decision-making

---

## 📊 Dashboard KPIs

- Total Patients
- Average Waiting Time
- Admission Rate
- Referral Rate
- Patient Satisfaction
- Department-wise Patients

---

## 📈 Dashboard Features

- Interactive Slicers
- Pivot Tables
- Pivot Charts
- KPI Cards
- Conditional Formatting
- Dynamic Reports

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Excel Formulas

---

## 📂 Dataset

The dataset contains hospital emergency records including:

- Patient ID
- Age
- Gender
- Admission Status
- Waiting Time
- Department
- Referral Status
- Visit Date
